# Security Policy

## Supported Versions

| Version | Supported |
|---------|----------|
| 5.x     | ✅ Yes   |
| 4.x     | ⚠️ Critical fixes only |
| < 4.0   | ❌ No    |

## Reporting a Vulnerability

Dacă ai descoperit o vulnerabilitate de securitate, **nu deschide un issue public**.

Trimite un email la: **security@[your-domain]** cu:
- Descrierea vulnerabilității
- Pașii pentru reproducere
- Impactul potențial
- (opțional) Sugestii de fix

Vei primi un răspuns în maxim **48 ore**. Dacă vulnerabilitatea este confirmată, va fi rezolvată și creditată în CHANGELOG.
